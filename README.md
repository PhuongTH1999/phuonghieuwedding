# 💒 Wedding Invitation Website

Trang web undang cưới cho Thanh Hiếu & Hoàng Phương

## 📋 Nội dung

- **Hero Section**: Ảnh nền carousel với thông tin cơ bản
- **Countdown**: Đếm ngược đến ngày cưới
- **Chi tiết sự kiện**: Lịch trình các sự kiện cưới
- **Album**: Thư viện ảnh cưới
- **Địa điểm**: Bản đồ và thông tin địa điểm tiệc cưới
- **RSVP**: Form xác nhận tham dự
- **Guestbook**: Sổ lưu bút từ khách mời
- **Gift Box**: Thông tin chuyển khoản quà cưới

## 🎨 Tính năng

- ✨ Responsive design (mobile, tablet, desktop)
- 🎵 Audio player cho nhạc nền
- 🎨 Color theme switcher (Gold, Rose, Sage, Dark)
- 📱 Mobile-friendly navigation menu
- 💫 Scroll animations
- 🖼️ Image lightbox gallery
- 📝 Guestbook with localStorage
- 📅 Countdown timer

## 🚀 Cách deploy lên Vercel

1. **Tạo GitHub repository**:
   ```bash
   git init
   git add .
   git commit -m "Initial commit: Wedding invitation website"
   git branch -M main
   git remote add origin https://github.com/your-username/your-repo.git
   git push -u origin main
   ```

2. **Deploy lên Vercel**:
   - Vào [vercel.com](https://vercel.com)
   - Kết nối GitHub account
   - Import repository
   - Deploy

## 📁 Cấu trúc thư mục

```
.

```

## 🎯 Lưu ý

- Tất cả ảnh đã được tối ưu hóa (.webp format)
- Form RSVP cần được cấu hình với Formspree hoặc email service khác
- Comments lưu trữ trên localStorage (không persistent trên server)
